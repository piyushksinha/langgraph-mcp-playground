# LangGraph MCP Playground

A hands-on learning project to build production-style AI agents using modern AI technologies.

## Technology Stack

- Python 3.13
- uv
- LangGraph
- LangChain
- MCP (Model Context Protocol)
- Ollama
- PostgreSQL
- ChromaDB
- Redis
- Langfuse
- Docker
- Rancher Desktop
- IBM OpenShift (later)
- IBM watsonx Orchestrate (later)
